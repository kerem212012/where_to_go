# WHERE TO GO
### How Need?
You don't know where you can go here with this map you can read descriptions about some places.
### How Run?
Play this code:
```pip install -r requirements.txt```
create file ```.env```
and write :
```python
SECRET_KEY = secretkey
ALLOWED_HOSTS = ["*",]
DEBUG = True
```
then play server:
```python
python manage.py runserver
```
